## 简介

MPax是使用Flutter开发的简约易用的音乐播放器。

## 特性

### 音频:

* [x] 播放音乐。
* [x] 读取音频元数据标签（部分支持）。
* [ ] 修改音频元数据标签。
* [x] 播放列表（初步支持）。
* [x] 搜索。

### 界面:

* [ ] 显示歌词。
* [x] 深色模式。
* [ ] 定时停止。
* [ ] 多视图（专辑，艺术家，文件夹...）。

### 平台：

* [x] 安卓。
* [ ] Windows。
* [ ] Linux。
